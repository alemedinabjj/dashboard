## 📝 Descrição 

Website para fazer upload de memes, ou imagens em geral.
Utilizando firebase(storage) como banco.


## 🚀 Tecnologias

Esse projeto foi desenvolvido com as seguintes tecnologias:

- ReactJS
- Javascript
- Firebase (storage, auth)
- Vite
- Phosphor Icons
- Styled Components
